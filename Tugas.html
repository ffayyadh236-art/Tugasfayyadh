Instruksi Command Prompt (CMD)

1. Navigasi File & Folder

  • dir: Menampilkan daftar file & folder yang ada di direktori saat ini.
  • cd [nama_folder]: Singkatan dari change directory. Digunakan untuk masuk ke folder.
  • cd ..: Kembali ke 1 tingkat folder sebelumnya (naik ke folder induk).
  • cls: Singkatan dari clear screen. Menghapus semua teks di layar CMD.

2. Pengelolaan File & Folder

  • mkdir [nama_folder]: Membuat folder baru.
  • ren [nama_lama] [nama_baru]: Mengubah nama file/folder.
  • copy [sumber] [tujuan]: Menyalin (copy) file ke lokasi tujuan.
  • del [nama_file]: Menghapus 1 atau lebih file.
  • rmdir [nama_folder]: Menghapus folder (folder tersebut harus kosong terlebih dahulu).

3. Informasi Jaringan

  • ping [alamat_website/IP]: Mengecek koneksi komputer agar bisa terhubung dengan alamat tertentu.
  • ipconfig: Menampilkan alamat IP pengguna.
  • tracert [alamat_tujuan]: Melacak rute perjalanan komputer menuju server tujuan.
  • netstat: Menampilkan koneksi yang aktif di komputer.

4. Informasi Sistem & Utilitas

  • systeminfo: Menampilkan spesifikasi hardware & versi Windows.
  • tasklist: Menampilkan daftar aplikasi yang sedang aktif.
  • taskkill /f /im [nama_aplikasi].exe: Menghentikan secara paksa suatu aplikasi.
  • sfc /scannow: Memeriksa & memperbaiki file sistem Windows yang rusak.

Cara Setting IP Address Statis

Panduan Konfigurasi Manual di Windows

1. Identifikasi Interface

Buka Command Prompt (CMD) dan ketik perintah berikut untuk melihat daftar kartu jaringan (Network Adapter) Anda:

ipconfig

Catat nama interface yang ingin diubah (misalnya: Ethernet atau Wi-Fi).

2. Membuka Control Panel

  • Tekan tombol Win + R, ketik ncpa.cpl lalu tekan Enter.
  • Klik kanan pada adapter jaringan Anda, pilih Properties.
  • Cari dan klik dua kali pada Internet Protocol Version 4 (TCP/IPv4).

3. Memasukkan Alamat IP

Pilih opsi "Use the following IP address" dan masukkan detail berikut:

  • IP Address: 192.168.1.10 (Contoh)
  • Subnet Mask: 255.255.255.0
  • Default Gateway: 192.168.1.1 (Biasanya IP Router)
Catatan: Pastikan IP yang Anda masukkan tidak sama dengan perangkat lain dalam satu jaringan agar tidak terjadi IP Conflict.

4. Verifikasi Hasil

Setelah klik OK, kembali ke CMD dan gunakan perintah di bawah ini untuk memastikan perubahan sudah diterapkan:

ipconfig /all

Panduan Teknologi Virtualisasi

Memahami Manfaat, Kegunaan, dan Konfigurasi Dasar

1. Oracle VM VirtualBox

Fungsi: Hypervisor tipe 2 (Hosted) yang berjalan di atas sistem operasi (Windows/Mac/Linux).

Manfaat: Gratis (Open Source), ringan, dan mendukung banyak OS tamu.

Cara Konfigurasi:
  1. Klik tombol New untuk membuat VM baru.
  2. Tentukan Nama, Tipe OS (ISO), dan alokasi Base Memory (RAM).
  3. Buat Virtual Hard Disk (VDI) dengan ukuran dinamis.
  4. Masuk ke Settings > Network, pilih Bridged Adapter jika ingin VM mendapat IP dari router.
  5. Klik Start untuk memulai instalasi.

2. VMware Workstation Pro/Player

Fungsi: Hypervisor tipe 2 kelas profesional dengan performa grafis dan stabilitas tinggi.

Manfaat: Fitur Snapshot yang canggih, integrasi drag-and-drop file antar OS, dan performa hardware yang optimal.

Cara Konfigurasi:
  1. Pilih Create a New Virtual Machine.
  2. Gunakan mode Typical untuk konfigurasi standar.
  3. Masukkan file ISO installer OS.
  4. Atur jumlah Core CPU dan RAM (disarankan minimal 2GB untuk OS modern).
  5. Selesaikan wizard dan jalankan VM untuk menginstal VMware Tools (penting untuk driver).

3. Proxmox Virtual Environment (VE)

Fungsi: Hypervisor tipe 1 (Bare-metal) berbasis Debian Linux. Berjalan langsung di hardware tanpa OS induk.

Manfaat: Skala enterprise, manajemen berbasis Web GUI, mendukung Container (LXC) dan Virtual Machine (KVM).

Cara Konfigurasi:
  1. Instal Proxmox langsung ke Server/PC menggunakan bootable USB.
  2. Setelah instal, akses panel kontrol melalui browser: https://[IP_ADDRESS]:8006.
  3. Upload file ISO ke menu Storage > ISO Images.
  4. Klik Create VM di pojok kanan atas.
  5. Konfigurasi Resource (CPU, RAM, Network) dan pilih ISO yang sudah diupload.
